How does it work?

0x53 48 41 4b 45 is a neural network written in Python with the Keras deep learning library using a TensorFlow backend, and trained on a specially formatted corpus of Shakespeare's sonnets. It is an LSTM network, a type of recurrent neural network especially well-suited to time-series prediction. We are using this network to generate text by feeding it a seed phrase, then repeatedly predicting the next most likely character and feeding the result back into the system. As you can see, even a very simple model without any pre-programmed natural language processing capabilities can achieve surprisingly coherent results, not only in word construction, but even in learning some high-level features of the sonnet form. This is why neural networks are so cool!

The web interface is built in Flask and hosted on Heroku.
